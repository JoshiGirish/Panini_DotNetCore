idfVector

Inverse Document-Frequency vector for the topic.

Declaration
public SortedDictionary<string, double> idfVector
Field Value
Type Description
System.Collections.Generic.SortedDictionary<System.String, System.Double>

This vector stores the IDF scores of each word of the lexicon that appears in the given topic. The higher the score of a word in the vector, the more unique is the word for the topic.

The length of the vector is equal to the maximum vocabulary size of the Lexicon.
