public int MaxVocabSize { get; set; }
Property Value
Type Description
System.Int32

Represents the maximum number of words allowed to be part of the lexicon. The n most popular words from all topic are extracted and used for TFIDF and Similarity score computation.

Important: The larger the MaxVocabSize the longer will be the TFIDF vectors and the longer it will take to compute the scores.
